### 📝 Repository Overview
이 저장소는 **빅데이터를 위한 파이썬** 강좌를 수강하며 학습한 내용을 정리하는 공간입니다. 파이썬의 기초 문법부터 시작하여 대량의 데이터를 효율적으로 처리하기 위한 데이터 구조, 제어문, 함수 및 실무 활용 능력을 기르는 실습 코드를 포함하고 있습니다.

### 📅 주차별 학습 요약 (Weekly Study)

| 주차 | 주요 학습 내용 | 핵심 키워드 |
| :--- | :--- | :--- |
| **Week 2** | 파이썬 환경 설정 및 기초 | 변수, 자료형, 기본 연산자 |
| **Week 3** | 시퀀스 자료구조 | 리스트(List), 튜플(Tuple), 인덱싱 및 슬라이싱 |
| **Week 4** | 조건문과 반복문 | if-else, for, while, range() |
| **Week 6** | 함수(Functions) | 사용자 정의 함수, 매개변수, 가변 인자 |
| **Week 7** | 데이터 맵핑 및 가공 | 딕셔너리(Dictionary), 집합(Set), 리스트 컴프리헨션 |
| **Week 8** | 심화 로직 설계 | 복합 제어문, 실무형 알고리즘 기초 |
| **Practice** | 날짜별 실전 실습 | 데이터 전처리 실습 (251114, 251121) |

### 🚀 주요 실습 및 과제 (Highlights)
* **Weekly Assignments:** 각 주차별 이론을 바탕으로 한 프로그래밍 과제 수행.
* **Data Processing Logic:** 딕셔너리와 리스트를 조합한 효율적인 데이터 관리 및 탐색 로직 구현.
* **Functional Programming:** 반복적인 데이터 처리를 위한 모듈화 및 함수 설계 실습.
* **Real-world Practice:** 실제 데이터셋 형태를 가정한 시뮬레이션 및 결과 도출 실습.

### 📂 폴더 구조 (Project Structure)
